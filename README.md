# DropSync Home Assistant Add-on Repository

This repository contains a custom Home Assistant add-on for DropSync.

## Install

1. Open Home Assistant.
2. Go to **Settings → Add-ons → Add-on Store**.
3. Open the three-dot menu and choose **Repositories**.
4. Add your Git repository URL.
5. Install **DropSync** from the store.

## Local testing

Copy this repository into the Home Assistant Supervisor add-ons directory and refresh the store.
